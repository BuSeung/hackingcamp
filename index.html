The difference between CTF and Bug bounty (10분) * CTF와 Bug Bounty의 이해 CTF와 Bug Bounty의 차이에 대한 설명 * 유명한 CTF 대회 소개 잘 알려진 CTF 이벤트 소개 * 버그 바운티 플랫폼 소개 인기 있는 Bug Bounty 플랫폼 소개 3. Hacking school sites and bug bounties (10분) * 학교 사이트 펜 테스팅하는 동기 설명 * 학교 사이트 취약점 종류 Directory listing wordpress plugin xss Credential stuffing to RCE * 버그 바운티 제보 취약점 제보를 위한 절차 4.
